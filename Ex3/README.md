# 🧠 Microprocessor Lab Experiment – EX3 (8086)

## Overview

This repository contains **Experiment 3: Simple 8086 Programs using BIOS & DOS Interrupts**, implemented using Assembly Language as part of academic coursework.

---

## Contents

### 🔷 8086 Microprocessor Experiments

* Printing character using BIOS interrupt
* String comparison using CMPSB
* File handling using DOS interrupt
* Pattern printing using BIOS interrupt

---

## 🧠 Concepts Used

* BIOS Interrupts (INT 10H)
* DOS Interrupts (INT 21H)
* String Instructions (CMPSB)
* Cursor Positioning
* File Handling Operations
* Register Manipulation

---

## 🔌 Tools Used

* EMU8086 Microprocessor Simulator
* DOSBox (optional simulation)
* 8086 Trainer Kit

---

## 📊 Programs Included

1. Print Character 'I'
2. String Compare Program
3. File Handling Program
4. Print 'A' Pattern Program

---

## 🎯 Learning Outcomes

* Understanding BIOS and DOS interrupts
* Displaying characters on screen
* Performing string comparison
* Implementing file operations
* Creating patterns using cursor control
* Improving low-level programming skills
