# 🧠 Microprocessor Lab Experiment – EX1 (8086)

## 📌 Overview

This repository contains **Experiment 1: Arithmetic Operations in 8086**, implemented using Assembly Language as part of the Microprocessor Lab coursework.

---

## 📂 Contents

### 🔷 Arithmetic Operations (8086)

* Addition of two 16-bit numbers
* Subtraction of two numbers
* Multiplication using MUL instruction
* Division using DIV instruction
* Expression evaluation

---

## 🧠 Concepts Used

* 8086 Assembly Language Programming
* Arithmetic Instructions (ADD, SUB, MUL, DIV)
* Carry and Borrow Handling
* Register Operations (AX, BX, CX, DX)
* Data Segment and Code Segment usage
* Looping and conditional jumps (JNC)

---

## 🔌 Tools Used

* EMU8086 Microprocessor Simulator
* DOSBox (optional)
* 8086 Trainer Kit

---

## 📊 Programs Included

1. Addition Program
2. Subtraction Program
3. Multiplication Program
4. Division Program
5. Expression Evaluation Program

---

## 🎯 Learning Outcomes

* Understanding arithmetic operations in 8086
* Learning how carry and borrow flags work
* Performing multiplication and division using registers
* Writing structured assembly programs
* Evaluating mathematical expressions using assembly logic
