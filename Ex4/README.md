# 🧠 Microprocessor Lab Experiment – EX4 (8086)

## Overview

This repository contains **Experiment 4: Interfacing and Programming LEDs using 8255 (7-Segment Display)**, implemented using Assembly Language as part of academic coursework.

---

## Contents

### 🔷 8086 Microprocessor Experiments

* Displaying single digit (0–F) using 7-segment display
* Displaying characters (H, E, L) using 7-segment display
* Interfacing 8255 Programmable Peripheral Interface
* LED control using output ports

---

## 🧠 Concepts Used

* 8086 Assembly Language Programming
* 8255 Programmable Peripheral Interface
* Port Addressing (OUT instruction)
* 7-Segment Display Encoding
* Delay Generation using loops
* Hardware Interfacing Techniques

---

## 🔌 Tools Used

* EMU8086 Microprocessor Simulator
* 8086 Trainer Kit
* 8255 Interfacing Module
* 7-Segment Display

---

## 📊 Programs Included

1. Single Digit Display using 7-Segment
2. HEL Display using 7-Segment

---

## 🎯 Learning Outcomes

* Understanding interfacing of 8255 with 8086
* Displaying digits and characters on 7-segment display
* Generating delays for hardware control
* Learning port-based input/output operations
* Practical knowledge of embedded interfacing
