# 🧠 Microprocessor Lab Experiment – EX2 (8086)

## Overview

This repository contains **Experiment 2: Sorting, Searching & String Operations in 8086**, implemented using Assembly Language as part of academic coursework.

---

## Contents

### 🔷 8086 Microprocessor Experiments

* Sorting of array elements (Ascending Order)
* Searching an element (Linear Search)
* Count the number of elements in an array
* Comparison of two strings
* Occurrence of a character in a string
* Substring search in a main string

---

## 🧠 Concepts Used

* 8086 Assembly Language Programming
* Array Traversal using SI Register
* String Instructions (CMPSB)
* Loop Control using CX Register
* Conditional Jumps (JZ, JNZ, JBE)
* Memory Addressing Techniques
* String and Array Processing

---

## 🔌 Tools Used

* EMU8086 Microprocessor Simulator
* DOSBox (optional simulation)
* 8086 Trainer Kit

---

## Learning Outcomes

* Understanding sorting techniques in assembly language
* Implementing linear search algorithm
* Performing string comparison operations
* Finding occurrence of characters in strings
* Implementing substring search logic
* Improving problem-solving using low-level programming
